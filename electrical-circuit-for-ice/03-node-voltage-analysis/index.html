Node-Voltage analysis

Step 1

Select a node as the reference node (Essential Node)
Assign V1,V2,V3,...,Vn1V_1,V_2,V_3,...,V_{n-1}V1,V2,V3,...,Vn1 to the remaining nonreference node

Step 2

Apply KCL to each of the n-1 nonreference nodes. User Ohms law to epress branch currents in terms of node voltages

Apply KCL to node #1 I1i1i2I2=0I_1 - i_1 - i_2 - I_2 = 0I1i1i2I2=0
Apply KCL to node #2 I2+i2i3=0I_2 + i_2 - i_3 =0I2+i2i3=0

i=VhigherVlowerR=VfromVtoRi = \frac{V_{higher} - V_{lower}}{R} = \frac{V_{from}-V_{to}}{R}i=RVhigherVlower=RVfromVto i1=V1R1,i2=V1V2R2,i3=V2R3i_1 = \frac{V_1}{R_1}, i_2 = \frac{V_1 - V2}{R_2}, i_3 = \frac{V2}{R_3} i1=R1V1,i2=R2V1V2,i3=R3V2

Substitude back to KCL equations

I1=I2+V1R1+V1V2R2I_1 = I_2 + \frac{V_1}{R_1} + \frac{V_1-V_2}{R_2} I1=I2+R1V1+R2V1V2 I2+V1V2R2V2R3=0I_2 + \frac{V_1-V_2}{R_2} - \frac{V_2}{R_3} = 0 I2+R2V1V2R3V2=0

Step 3

Solve the remianing simultaneous equations to find unknown node voltages

Notes

  1. All 3 Laws are used
  2. KCL is used at each nonrefernce node
  3. Ohms law is used to determine current in branches
  4. KVL is used to determine the voltage drop across resistors
