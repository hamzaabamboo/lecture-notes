Difference between branch current and mesh current

I1,I2,I3I_1,I_2,I_3I1,I2,I3 are branch currents

V3=i3R3 V_3 = i_3 \cdot R_3 V3=i3R3

i1,i2i_1, i_2i1,i2 are node currents
throuch R3R_3R3 there are two mesh currents

V3=(i1i2)R3 V_3 = (i_1 - i_2) \cdot R_3 V3=(i1i2)R3

Mesh Analysis

3 Steps

1. Assign mesh currents. i1,i2...ini_1,i_2...i_ni1,i2...in

2. Apply KVL to each mesh. Use Ohms law to express voltages in terms of mesh currents

Apply KVL to mesh 1

V1+i1R1+(i1i2)R3=0 -V-1 + i_1R_1 + (i_1-i_2)R_3 = 0 V1+i1R1+(i1i2)R3=0

Apply KVL to mesh 2

(i2i1)R3+i2R2+V2=0 (i_2-i_1)R_3 + i_2R_2 + V_2 = 0 (i2i1)R3+i2R2+V2=0

3. Solve the simultaneous equations

Special Case 2 : Super mesh

Supermesh: Comine meshes 1 and 2 into a super mesh
Important note: A supermesh does not have current of its own. (Well just use i1i_1i1 and i2i_2i2)

Apply KVL to the supermesh:

i1+2(i1i3)+4(i2i3)+10=0 i_1 + 2(i_1-i_3) + 4(i_2-i_3) + 10 = 0 i1+2(i1i3)+4(i2i3)+10=0

Apply KVL to mesh 3:

3i3+2(i3i1)+4(i3i2)=0 3i_3 + 2(i_3-i_1) + 4(i_3-i_2) = 0 3i3+2(i3i1)+4(i3i2)=0

Apply KCL to node a (our third magical equation):

i2i1=5 i_2 - i_1 = 5 i2i1=5
